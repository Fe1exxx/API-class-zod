# TypeScript API Client
## Типобезопасный API-клиент с валидацией данных на основе TypeScript, Zod.
Современный HTTP-клиент для работы с REST API, который обеспечивает:
 Типобезопасность - полная типизация ответов через TypeScript дженерики
 Валидация данных - проверка структуры ответов через Zod схемы
 Обработка ошибок - try/catch
 Чистая архитектура - инкапсуляция, разделение ответственности, SOLID принципы
